기본 속성 선택자

기본 선택자 뒤에 [] 를 사용하여 속성과 속성값을 사용한다.
(주로 id나 class 속성이 아닌것들에 사용된다.)
선택자[속성=속성값] {설정내용} 이런형식으로 사용된다.

후손 선택자와 자손 선택자

자손선택자 : 바로 아래의 요소
후손선택자 : 하위 요소 모두
자손선택자 사용 : 선택자 > 자손선택자 {설정내용} 의 형식
후손선택자 사용 : 선택자 후손선택자 {설정내용} 의 형식

동위 선택자

동위 관계(형제 관계)에서 뒤에 위치한 태그를 선택할 때 사용
바로 뒤의 요소 선택시
: 선택자 a + 선택자 b {설정내용} 형식
바로 뒤에있는 요소를 모두 선택
: 선택자 a ~ 선택자 b {설정내용} 형식

반응 선택자

사용자의 움직임에 따라 달라지는 선택자
사용자가 클릭한 태그 : 선택자:active {설정내용}
사용자의 마우스가 올라가있는 태그 : 선택자:hover {설정내용}

active 테스트 hover 테스트

상태 선택자

입력 양식의 상태에 따라 선택되는 선택자
체크 상태의 input 태그 선택 : input태그:checked{설정내용}
